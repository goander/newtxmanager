# Finance Tracker (v2)
Vite + React + Tailwind. Ready for Netlify.

## Local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Netlify
- Drag & drop the `dist/` folder to Netlify **Deploy manually**
- or connect your Git repo (Build cmd: `npm run build`, Publish dir: `dist`)
